### 3.2 注册步骤
1.  打开 **ChatGPT 官网入口** [https://chat.openai.com/auth/login](https://chat.openai.com/auth/login)。
2.  点击 **Sign Up**（注册）。
3.  输入邮箱地址并设置密码，点击 Continue。
4.  登录邮箱查收验证邮件，点击 Verify email address。
5.  填写姓名和生日（需年满18岁）。
6.  在手机号验证页面，选择对应的国家，输入从接码平台获取的号码。
7.  输入收到的 6 位数验证码。
8.  注册成功！

### Q1: 手机上怎么用 ChatGPT？有官方 APP 吗？
**A**: **OpenAI 官网** 推出了 iOS 和 Android 版的官方 APP，但需要登录美区 Apple ID 或 Google Play 商店下载，且同样需要稳定的网络环境。国内用户也可以通过浏览器访问第三方服务，手机端通常也有良好的适配。

### Q2: 为什么我的 ChatGPT 账号被封了？
**A**: 常见原因包括：使用被污染的批量注册 IP、使用虚拟信用卡充值 Plus 被拒付、频繁切换不同国家的 IP 节点。

**A**: 这取决于具体的服务是否接入了 DALL·E 3 模型。GPT-4o 已支持图像生成功能，部分第三方服务也集成了此能力。

### Q4: 现在的 ChatGPT 4.0 是免费的吗？
**A**: 在 **ChatGPT 官网**，免费用户可以有限次使用 GPT-4o，但 GPT-4 仍主要面向 Plus 会员。部分第三方服务会提供每天一定次数的免费调用额度。

### Q5: 使用第三方服务安全吗？
**A**: 建议选择有明确运营主体、隐私政策透明的服务。避免使用要求提供过多个人信息或来源不明的平台。
